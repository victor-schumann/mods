# Silksong Mod - Alphabetical Order Items

**Author:** ivorydev  
**Version:** 1.0.0  
**Nexus Page:** https://www.nexusmods.com/hollowknightsilksong/mods/806

## Description
sorts `UISelectionList` items alphabetically. updates list and hierarchy.

## Installation
copy `AlphabeticalOrderItems_BellwaysVentrica.dll` to `BepInEx/plugins` after installing BepInEx and running once.

## Usage
mod works automatically. f5 can be pressed to check if loaded.

## Compatibility
silksong, bepinex, `UISelectionList` with `TMP_Text`.
